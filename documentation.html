Art-Folio Documentation

Creative Portfolio & Agency Landing Page Template

Version 1.0.0 | Last Updated: July 18, 2025

Introduction

Thank you for purchasing Art-Folio, a modern landing page template designed for designers, creatives, and agencies. Built with Next.js, Tailwind CSS, and GSAP, Art-Folio offers a fast, responsive, and animated portfolio showcase.

This documentation guides you through installing, customizing, and deploying the support. For support, visit support.teachfosys.com.

Features

  • Built with Next.js for fast performance and server-side rendering
  • Styled with Tailwind CSS for easy customization
  • Smooth animations using GSAP and ScrollTrigger
  • Fully responsive for mobile, tablet, and desktop
  • Light and dark mode support
  • Perfect for portfolios, agency websites, or creative showcases
  • Ready to deploy on Vercel or similar platforms

Getting Started

Requirements

Before you begin, ensure you have:

  • Node.js (version 16 or higher) - Download from nodejs.org
  • npm (included with Node.js) or yarn (optional)
  • A code editor like Visual Studio Code

Installation Steps

  1. Unzip the Template:

    Extract the purchased ZIP file to a folder on your computer.

  2. Install Dependencies:

    Open a terminal, navigate to the project folder, and run:

    npm install
  3. Start the Development Server:

    Run the following command to start the template:

    npm run dev

    Open http://localhost:3000 in your browser to view the support.

Project Structure

The template is organized as follows:

art-folio/
├── components/         # Reusable UI components
├── pages/             # Next.js page routes
├── public/            # Images, favicon, and static assets
├── styles/            # Global CSS and Tailwind styles
├── tailwind.config.js # Tailwind CSS configuration
├── next.config.js     # Next.js configuration
├── package.json       # Project dependencies and scripts
├── README.md          # Basic project information
└── documentation.html # This file

Customization

Customize Art-Folio to match your brand:

  • Colors: Edit tailwind.config.js to change the color scheme.
  • Fonts: Modify styles/globals.css to use a different font (DM Sans is the default).
  • Animations: Update GSAP animations in components/ files.
  • Content: Replace images in public/ and text in components/ or pages/.

Refer to the Tailwind CSS documentation and GSAP documentation for advanced customization.

Deployment

To deploy Art-Folio to a live website:

  1. Create a repository on GitHub and push your project files.
  2. Sign up at Vercel and connect your GitHub repository.
  3. Use the default Next.js build settings in Vercel.
  4. Deploy your site and share the live URL.

For other hosting platforms, refer to their documentation for Next.js deployment.

Troubleshooting

  • npm install fails: Ensure Node.js is installed and run npm cache clean --force, then retry.
  • Page not loading: Check that the development server is running (npm run dev) and visit http://localhost:3000.
  • Assets missing: Ensure all images and files are in the public/ folder.
